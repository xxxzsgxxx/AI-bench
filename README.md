# AI-bench

硬件性能测试与自动化基准测试工具集

## 功能特性
- 硬件信息采集（CPU/GPU/内存）
- 自动化性能基准测试
- 测试报告生成
- 系统资源监控

## 快速开始
```bash
git clone https://github.com/your-account/AI-bench.git
cd AI-bench
chmod +x *.sh
./install.sh
./main.sh
```

## 目录结构
```
├── modules/          # 功能模块
├── examples/         # 使用示例
├── sysinfo/          # 系统信息采集
├── docs/             # 文档
└── .github/workflows # CI/CD配置
```
